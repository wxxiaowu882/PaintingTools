## 一、核心概念

**基础规律**讲「为什么」会亮会暗；**万物观察法**讲「怎么看」——从大到小，**先势后质**。  
（万物观察法用阴阳作**观察比喻**，帮助记忆；写生仍以眼中所见为准。）

### 【基础规律】面转则明暗分，体积现

面的转折 → 受光变 → 明暗分 → 体积现  
<典语>：**圆转缓变 / 方折突变**（曲而渐、折而断）

### 【万物观察法】

**一生二 · 亮暗分**  
世界有**阴阳两面**：先把物体分成 **亮部**（向光）、**暗部**（背光）。

**二生三 · 交界立**  
阴阳两面的**交锋就是「三」**（即 **明暗交界带**，不是一条线）：在亮、暗之间找这条界；**大转折**（棱、曲面转向）决定界在哪里、怎么转。

**三生万物 · 层次与加减**  
交锋处融合、分化，形成**介于阴阳之间**的各种状态：

- **亮部分化**：
  - **高光**、
  - **亮灰**（哑光可无单独高光，整块亮部即可）
- **暗部分化**
  - **暗灰**
  - **反光**（多在 **暗侧根部以上** 的中下段，易接到台面/环境反射；紧贴根部往往更暗）
  - **投影**（身外，从 **暗侧** 读）
  - **<最深处>**：
    - **交界深带**、
    - **暗侧物体根部**（暗部靠台面的一截，夹角闭塞，直射与反光都难进）、
    - **投影根部**（投影挨物体脚、挨桌面处；常与暗侧根部连成最深带）
- **万物·推拉**（**结构读对后再做**）：固有色 · 材质 · 离光源远近 · 观察角度 · 空间介质——只**推亮或压暗**，不改「面转则明暗分」。（**观察角度**多在「看之前」定好；换角度等于重做亮暗分。）

<典语>：**两仪渐分** / **执两端而用中** / **五息相承**（细部一层接一层，别切成补丁）

**由亮→暗**（核对用，**不是**观察第一步）：  
高光 ＞ 亮灰 ＞ 反光 ＞ 暗灰 ＞ 交界深带；**全图最深**常在 **暗侧物体根部** 与 **投影根部**（光位不同时，也与交界深带争最深）。

### 【先势后质 · 走法】


|       | 步骤        | 做什么                 |
| ----- | --------- | ------------------- |
| **势** | 一生二 + 二生三 | 先 **亮/暗**，再找 **交界** |
| **质** | 三生万物      | 再拆细，**最后推拉**        |


1. **一生二** — 分 **亮部、暗部**
2. **二生三** — 找 **明暗交界** 与 **大转折**
3. **万物·形** — **亮部分化**；**暗部分化**（层次 + **最深**）
4. **万物·推拉** — 固有色、材质、远近、环境等 **加减**（不改结构）

**顺序反了**（先推拉或先抠高光）→ 易 **糊、花、喧宾夺主**。  
<典语>：**先势后质**

## 附 · 速查


| 说法       | 指什么                          |
| -------- | ---------------------------- |
| **一生二**  | 亮部 / 暗部                      |
| **二生三**  | 明暗交界带 + 大转折                  |
| **三生万物** | 亮部分化 / 暗部分化（层次 + **最深**）/ 推拉 |
| **先势后质** | 势=一二；质=万物（推拉最后）              |


常见误会：**反光**别画比 **亮灰** 还亮；**推拉**不能代替 **亮暗分**。**最深**只看 **暗侧**，不是亮侧贴地一带。
